多选下拉选择

模拟select元素的组件


单选框

v-component="select"

<div src="json/mult-select.json" placeholder="多选" v-component="mult-select" v-with="value:price"></div>

参数

src(string) - 获取数据的地址。
placeholder(string) - 提示信息。
single(bool) - 是否单选。默认值为false
value - 选中值。

使用 v-with 同步参数


Example

当前选中：{{mult}}
当前选中：{{single}} 
<form>
    <div class="form-group">
        <div src="json/mult-select.json" class="form-control" style="width:260px; display:inline-block;" placeholder="mult select" v-component="mult-select" v-with="value:mult"></div>
        当前选中：{{mult}}
    </div>
    <div class="form-group">
        <div src="json/mult-select.json" class="form-control" style="width:160px; display:inline-block;" single="true" placeholder="single select" v-component="mult-select" v-with="value:single"></div>
        当前选中：{{single}}
    </div>
</form>
