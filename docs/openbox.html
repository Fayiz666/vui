Openbox

弹出层组件


vui.openbox

vui.openbox({
    show: true,
    width: 8,
    src: url,
    title: 'Openbox',
    btns: ['ok', 'close'],
    data: { 
        model: vui.utils.copy(this.model)
    },
    callback: function (model) {
        this.model = model
    }.bind(this)
})

参数

show - 是否立即打开，默认为false。
width - 宽度(值为1-12)，默认值：6。使用.col-md-*
src - 页面内容地址。
title - 显示的title。
btns - 页脚显示按钮，'ok', 'close'为内置按钮，如果要自定义，按照{ text: '按钮', type:'default', fn: function }格式添加。
data - 要传递的数据，如果传递的是object，并且不希望值被修改，使用copy方法clone一个object。
callback - 回调函数，目前只回传openbox的data.model。
