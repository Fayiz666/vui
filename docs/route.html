Route

用来处理url route。目前在监听 hashchange 这个事件，html5Mode 下不会触发这个事件，后面再修改。

route

route(fn, init)

根据当前url处理component

参数

fn - 回调函数。
init - 初始化时是否执行此方法，默认为false

vui.route(function (path) {
    if (path.charAt(path.length-1) !== '/')
        vui.$data.view = path

    // 延时等component 执行完
    setTimeout(function () {
        prettyPrint()
    }, 500)
}, true)

route.bind

route.bind(fn, basepath)

绑定一个route（hashchange）事件，使用了一个hashCode的方法来做handle

参数

fn - 回调函数，如果事件已存在，忽略。
basepath - 值为true时，只匹配path部分，默认为false。

var fn = function () { 
    // 做点什么
}
vui.route.bind(fn, true)

route.unbind

route.unbind(fn)

移除一个route（hashchange）事件

参数

fn - 函数，如果为空，删除所有绑定事件。

var fn = function () { 
    // 做点什么
}
vui.route.unbind(fn)

route.getComponent

route.getComponent(path, fn)

获取指定地址模板，转为component，如果已存在，从缓存中获取。

参数

path - 模板页面地址，component name。
fn - 加载成功后回调方法。

var fn = function () { 
    // 做点什么
}
vui.route.getComponent(fn)
