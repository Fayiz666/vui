Checkbox 和 Radio

单选和多选组件，合并为option


v-component="option"

<div type="checkbox|radio" v-component="option" v-with="value:value" src="" options="json"></div>

参数

type(string) - 类型，"checkbox" 或者 "radio"。
src(string) - 可选。数据源地址。
options(string) - 可选。json格式数据，当此项不为空时，忽略src。

数据格式

json格式，key为选项显示内容，value为值。 

{
    "选项一": "options 1",
    "选项二": "options 2",
    "选项三": "options 3",
    "选项四": "options 4"
}
