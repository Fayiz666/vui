Checkbox 和 Radio

单选和多选组件


v-component="option"

<div v-component="radio" v-with="value:value" src="" options="json"></div>
<div v-component="checkbox" v-with="value:value" src="" options="json"></div>

参数

src(string) - 可选。数据源地址。
options(string) - 可选。json格式数据，当此项不为空时，忽略src。

数据格式

json格式，text为选项显示内容，value为值。 

[
    {"text": "选项一", "value": "options 1"},
    {"text": "选项二", "value": "options 2"},
    {"text": "选项三", "value": "options 3"},
    {"text": "选项四", "value": "options 4"}
]


Example Options


radio: {{ra1}}
checkbox: {{cb1}} 
<div v-component="radio" v-with="value:ra1" options="[{text:'option 1', value:1}, {text:'option 2', value:2}]"></div>
<div v-component="checkbox" v-with="value:cb1" options="'option 1':1,'option 2':2,'option 3':3"></div>

<button style="button" v-on="click:ra1=2" class="btn btn-default">set radio 2</button>
<button style="button" v-on="click:cb1=[2,3]" class="btn btn-default">set checkbox 2,3</button>
