日期选择

日期选择控件

v-component="date"

<div class="form-control date" placeholder="日期选择" unixtime="true" v-component="date" v-with="value:date"></div>

参数

placeholder(string): 相当于input的placeholder属性。
unixtime(bool): 默认值false，当为true时，date的值为unixtime形式。
value - 选中值。使用 v-with 同步参数


Example



当前选择日期：{{scope.date1}} - {{scope.date2}}

<div class="form-control date" v-component="date" v-with="date:scope.date1"></div>
<div class="form-control date date-picker-up" placeholder="unixtime" unixtime="true" v-component="date" v-with="date:scope.date2"></div>
<br />
<p>当前选择日期：{{scope.date1}} - {{scope.date2}}</p>
