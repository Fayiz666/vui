在superagent上修改了一点，基础的文档参考http://visionmedia.github.io/superagent/。

header里增加了Authentication验证，当服务器header返回Authentication时，会记录下来并在下次发送请求时在header里加上Authentication。

getTemplate

在superagent外新增的方法，获取模板页，如果模板内容已请求过，从缓存中获取。

参数

string - 请求地址

返回值

object - 根据缓存中是否已有模板返回 Template 或 TemplateRequest 对象，拥有共同方法 end。
end 参数为 function(template)，template 为模板内容，取值为 res.text 

request.getTemplate('/api/template')
    .end(function(template) {
        document.body.innerHTML = template
    })
