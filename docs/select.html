下拉选择组件

模拟select元素的组件


单选框

v-component="select"

<div src="json/select.json" cache="false" placeholder="select price" v-component="select" v-with="value:price"></div>

参数

src - 获取数据的地址。（特殊值"bool"，自动生成["","是","否"]三个选项）。
placeholder - 提示信息。
cache(bool) - 是否缓存数据，默认值为true
value - 选中值。

使用 v-with 同步参数


Example

X
= {{scope.price * scope.count}} 
<form class="form-inline" role="form">
    <div class="form-group">
        <div src="json/select.json" class="form-control" style="width:160px" placeholder="select price" v-component="select" v-with="value:scope.price"></div>
    </div>
    <div class="form-group">
        <span> X </span>
    </div>
    <div class="form-group">
        <div src="json/select.json" class="form-control" style="width:160px" cache="false" placeholder="select count" v-component="select" v-with="value:scope.count"></div>
    </div>
    <div class="form-group">
        <span> = {{scope.price * scope.count}}</span>
    </div>
</form>
