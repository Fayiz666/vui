下拉选择组件

模拟select元素的组件


单选框

v-component="select"

<div src="json/select.json" placeholder="select price" v-component="select" v-with="value:price"></div>

参数

src - 获取数据的地址。
placeholder - 提示信息。
value - 选中值。

使用 v-with 同步参数


Example

X
= {{price * number}}
