Tree View

tree

v-component="tree"

<ul src="json/tree.json" selectable="true" key="id" v-component="tree"></ul>

参数

src(string) - 可选。数据源地址。
selectable(bool) - 默认值为false，为true时可以选择。
key(string) - 获取当前选中元素的值，默认key为'id'。


Example

    selected id: {{scope.value}}
