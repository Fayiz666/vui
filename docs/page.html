数据显示及分页

页面显示组件。


数据展示

v-component="page"

<ELEMENTTAG src="json/tablelist.json" delay="true" pageable="false" routeChange="true" size="10" v-component="page">

参数

src - 获取数据的地址。
delay - 数据是否延时加载，默认为false。
routeChange - 筛选时是否改变route值，默认为false。
size - 每页返回数量，默认值20。
pageable - 是否显示分页，默认值true。


分页

v-component="pagination"

<ELEMENTTAG v-component="pagination" v-with="page:pager.page, size:pager.size, total:total">

参数

page - 当前页码，默认值为1。
size - 每页数据数量，默认值为20。
total - 记录总数。

使用 v-with 同步参数

page component 默认包含 pagination
